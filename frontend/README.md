# Student Management System - Frontend

A Flutter application for managing student attendance, reports, and administration.

## Getting Started

1. Ensure Flutter is installed on your system.
2. Navigate to the frontend directory.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the application.

## Project Structure

- `lib/core/`: Common system logic (config, network, storage, utils)
- `lib/shared/`: Reusable UI components (widgets, theme)
- `lib/modules/`: Feature modules (auth, admin, student, parent, attendance, report, class_management, permission)

## Troubleshooting

- If you encounter build errors, ensure your Flutter version is compatible (SDK >=3.0.0 <4.0.0).
- Run `flutter doctor` to check for any issues with your Flutter setup.
